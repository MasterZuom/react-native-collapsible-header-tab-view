# 贡献指南

感谢你对 react-native-collapsible-header-tab-view 的关注！我们欢迎所有形式的贡献。

## 开始贡献

### 1. Fork 和 Clone

```bash
# Fork 项目到你的账户
git clone https://github.com/your-username/react-native-collapsible-header-tab-view.git
cd react-native-collapsible-header-tab-view
```

### 2. 安装依赖

```bash
npm install
# or
yarn install
```

### 3. 创建分支

```bash
git checkout -b feature/your-feature-name
```

### 4. 提交更改

```bash
git add .
git commit -m "feat: add your feature description"
```

### 5. Push 并创建 Pull Request

```bash
git push origin feature/your-feature-name
```

## Commit 消息规范

请遵循 [Conventional Commits](https://www.conventionalcommits.org/) 规范：

- `feat:` 新功能
- `fix:` 修复 bug
- `docs:` 文档更新
- `style:` 代码风格调整（不影响功能）
- `refactor:` 代码重构
- `perf:` 性能优化
- `test:` 添加或修改测试
- `chore:` 其他更改（如依赖更新）

## 报告 Bug

请通过 [GitHub Issues](https://github.com/MasterZuom/react-native-collapsible-header-tab-view/issues) 报告 bug，包含：

- Bug 描述
- 复现步骤
- 期望行为
- 实际行为
- React Native 版本
- 设备/模拟器信息

## 功能请求

有新的功能想法？请提交 GitHub Issue 讨论！

## 许可证

贡献代码时，你同意你的贡献将在 MIT 许可证下发布。
